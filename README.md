# بوت ألعاب ديسكورد — روليت / مافيا / بومب / نرد 🎮

بوت واحد فيه 4 ألعاب + نظام نقاط ومتجر، محمي برتبة معينة.

## 🔒 الحماية بالرتبة

البوت مقفول بالكامل على رتبة **Games** (الايدي: `1517638794604707962`).
أي شخص ما عنده هذي الرتبة، لو حاول يستخدم أي أمر أو يضغط أي زر، يوصله رد:
> ❌ هذا الأمر مخصص فقط لأصحاب رتبة Games.

تقدر تغيّر الايدي من ملف `utils/config.js`.

⚠️ **مهم:** البوت يحتاج صلاحية **GuildMembers Intent** مفعّلة من Developer Portal حتى يقدر يشوف رتب الأعضاء (موضح بالأسفل).

---

## 📦 التثبيت المحلي (قبل الرفع)

```bash
npm install
```

انسخ `.env.example` وسمّه `.env` وعبّي:
```
TOKEN=توكن البوت
CLIENT_ID=Application ID
GUILD_ID=ايدي السيرفر
```

سجّل الأوامر (مرة واحدة، أو بعد أي تعديل):
```bash
npm run deploy
```

شغّل البوت محلياً للتجربة:
```bash
npm start
```

## ⚙️ إعداد Discord Developer Portal (ضروري)

1. روح [discord.com/developers/applications](https://discord.com/developers/applications)
2. افتح تطبيقك > تبويب **Bot**
3. فعّل:
   - ✅ **SERVER MEMBERS INTENT** (ضروري لفحص الرتبة)
   - ✅ **MESSAGE CONTENT INTENT** (ضروري لقراءة إجابات لعبة البومب بالشات)
4. تأكد إن البوت مدعو للسيرفر بصلاحيات: `Send Messages`, `Use Slash Commands`, `Embed Links`, `Attach Files`, `Add Reactions`

---

## 🚀 الرفع على Railway

1. روح [railway.app](https://railway.app) وسجّل دخول (يفضل GitHub)
2. **New Project > Deploy from GitHub repo** (لازم ترفع هذا المجلد كـ repo على GitHub أول)
   - أو **Empty Project** ثم اسحب الملفات يدوياً عبر Railway CLI
3. بعد إنشاء المشروع، روح لتبويب **Variables** وضيف:
   - `TOKEN`
   - `CLIENT_ID`
   - `GUILD_ID`
4. Railway يكتشف `package.json` ويشغل `npm start` تلقائياً (موجود ملف `railway.json` يثبت هذا الإعداد)
5. **مهم:** قبل أول تشغيل، شغّل أمر تسجيل السلاش مرة واحدة فقط. أسهل طريقة:
   - افتح تبويب **Settings > Deploy** في Railway واستخدم زر "Run Command" (لو متوفر)، أو
   - شغّله محلياً من جهازك مرة واحدة بنفس التوكن: `npm run deploy`
   (تسجيل الأوامر لا يحتاج يكون من نفس مكان تشغيل البوت، فقط يحتاج نفس TOKEN/CLIENT_ID/GUILD_ID)
6. بعد أول Deploy، تأكد من الـ Logs إنه طبع: `✅ البوت شغّال: BotName#0000`

البوت بعدها يفضل شغال 24/7 على Railway بدون ما يحتاج جهازك يكون مفتوح.

## 🎮 الأوامر والألعاب

### 🎰 `/روليت [رهان]`
- روليت جماعي حتى 10 لاعبين، كل واحد يدخل بنفس قيمة الرهان (افتراضي 50 نقطة).
- 30 ثانية للانضمام، بعدها المنشئ يضغط "دوّر الحين" أو ينتهي الوقت تلقائياً.
- العجلة تدور وتوقف على فائز واحد ياخذ كل النقاط (Pot).
- يحتاج لاعبين اثنين فأكثر.

### 🕵️ `/مافيا`
- لعبة مافيا جماعية كلاسيكية (مافيا/محقق/طبيب/مواطن).
- أدوار ليلية بالخاص + تصويت نهاري بالقناة، حتى تفوز المافيا أو المواطنين.

### 💣 `/بومب`
- لوبي انضمام، ثم "ابدأ اللعبة".
- كل جولة حرفين (مثلاً ق-ن)، حامل القنبلة يكتب كلمة بالشات تبدأ وتنتهي بنفس الحرفين قبل ينتهي الوقت.
- إجابة صحيحة → يمرر القنبلة لأي لاعب بزر.
- ينتهي الوقت بدون إجابة = ينفجر ويخرج. آخر لاعب باقي = الفائز.

### 🎲 `/نرد [الخصم] [الهدف]`
- نرد بالتبادل بين لاعبين. كل دور: نرد أساسي + نرد معدِّل (زي +6 أو -1).
- تثبيت أو إعادة رمي مرة وحدة بالدور.
- أول من يوصل للهدف (افتراضي 15) يفوز.

### 💰 `/نقاطي`
- يعرض نقاطك، انتصاراتك، خسائرك، وعدد الحمايات/النيوك/الهيكر اللي عندك.

### 🏆 `/توب`
- أفضل 10 لاعبين بالسيرفر حسب النقاط.

### 🛒 `/متجر`
- شراء: 🛡️ حماية (150) — ☢️ نيوك (300) — 💻 هيكر (250)
- النيوك يسرق 50% من نقاط لاعب تختاره (إلا إذا كان عنده حماية).
- زر "حقيبتي" يعرض إيتماتك.

## 🗄️ تخزين البيانات

البيانات (النقاط، الإيتمات...) تُخزّن بملف `data/db.json` تلقائياً — ما يحتاج قاعدة بيانات خارجية.
⚠️ على Railway، إذا أعدت الـ Deploy فممكن يصفّر الملف إذا مو مربوط بـ Volume. لو تبي البيانات تبقى دائمة، أضف **Volume** من Railway وربطه بمجلد `data/`.
